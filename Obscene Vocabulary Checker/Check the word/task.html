Description

In addition to the previous stage, you need to determine if a word is taboo.

Objectives
  • Read the name of the file that contains taboo words from the user input;
  • Read a file with taboo words;
  • Read one word from the input. If the word is a taboo one, print True, otherwise False.

The program should be case-insensitive!

Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

The words.txt contents:
disgusting
unpleasant
ugly
bad

Example 1:

> words.txt
> Bad
True

Example 2:

> words.txt
> Badge
False

Example 3:

Entering a blank line

> words.txt
> 
False
