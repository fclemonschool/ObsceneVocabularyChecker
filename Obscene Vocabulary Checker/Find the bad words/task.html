Description

In this stage, your program should be able to read a file that contains taboo words, go through this file, and print those words. List one word on each line.

Objectives

In this project, we will assume that a word is a sequence of Latin characters of lower or upper case.

  • Read the name of the file that contains taboo words from the user input;
  • Read the file;
  • Print each word on a separate line.
Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1:

The forbidden_words.txt contents:
disgusting
unpleasant
ugly
bad

> forbidden_words.txt
disgusting
unpleasant
ugly
bad

Example 2:

the forbidden_words.txt contents:
an empty file

> forbidden_words.txt
